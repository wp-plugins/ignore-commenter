== Description ==

Adds a button to your blog post to allow users to ignore (or hide) comments from unwanted commenters. It's a good way to limit abuse of your blog post comments. No special settings required. Just activate and use.

To ensure that all commenters are tracked by their user id, you must restrict commenting to only logged-in users.


Major features in Ignore Commenter include:

* Automatically places a button on the comments on your website or blog.
* Contains admin page for each user to view all ignored commenters.

== Installation ==

Upload the Ignore Commenter plugin to your blog, Activate it, that's it!

1, 2, 3: You're done!
